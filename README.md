## Requirements

- iOS 10.0+

## Installation

### Cocoapods

Simply add the following line to your Podfile:

```ruby
pod 'SwiftEssentialsKit'
```

The default submodules are:

- SwiftEssentialsKit/PackageDTO
- SwiftEssentialsKit/AlertControllerBuilder
- SwiftEssentialsKit/Convenient
